# ccwc
A command line tool that counts bytes, newlines and words in a text file

### usage
```
ccwc <file path> <option>
```

#### options
-c counting bytes

-l counting newlines

-w counting words

[codingchallenges.fyi](https://codingchallenges.fyi/challenges/challenge-wc/)
